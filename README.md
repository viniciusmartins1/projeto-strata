# Projeto Strata

Projeto desenvolvido para prática do curso de desenvolvimento web na [Udemy](https://www.udemy.com). 

**Durante o projeto foram abordados conceitos como:**
- `Formulários;`
- `Tags semânticas;`
- `Grids simples;`
- `Posicionamento fixed;`
- `Classes gerais;`
- `Icones como fonte;`
